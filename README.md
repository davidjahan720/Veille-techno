# Tech Watch Web App

A complete technological watch web application with a daily AI-generated summary and email notifications.

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express (hosted as Vercel Serverless Functions)
- **AI**: Mistral API (mistral-small)
- **Email**: Resend
- **Automation**: Vercel Cron Jobs

## Features

- **Dashboard**: View latest articles from multiple RSS sources categorized into Frontend, Backend, Cloud & DevOps, and RGPD & Souveraineté.
- **AI Summary**: Daily summary of top articles per category generated by Mistral AI.
- **Email Notifications**: Daily email sent at 8 AM UTC with the AI-generated summary.
- **Responsive Design**: Clean and professional UI that works on mobile and desktop.

## Local Development

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Environment Variables**:
   Create a `.env.local` file with the following keys:
   ```env
   MISTRAL_API_KEY=your_mistral_api_key
   RESEND_API_KEY=your_resend_api_key
   NOTIFICATION_EMAIL=your_email@example.com
   VITE_API_URL=/api
   ```

3. **Run the app**:
   ```bash
   npm run dev
   ```

## Deployment to Vercel

1. **Push to GitHub**:
   Create a repository and push your code.

2. **Connect to Vercel**:
   - Go to [Vercel](https://vercel.com) and import your repository.
   - Add the Environment Variables in the project settings.
   - Deploy!

3. **Cron Job Configuration**:
   The cron job is already configured in `vercel.json`. It will run automatically once deployed. You can also manually trigger it by visiting `https://your-app.vercel.app/api/cron`.

## RSS Sources

The app fetches news from various sources including Frontend Masters, web.dev, Hacker News, High Scalability, InfoQ, AWS Blog, CNIL, ANSSI, and more.
